# TasmoBackup
Backup the configs of all your Tasmota devices

# Features
* Backup single devices
* Backup all devices
* Remove devices
* Download individual backups

# Requirements

* PHP
* Mysql / MariaDB - Import tasmobackup.sql
* Writeable directory named "backups" in the same folder as the index.php
